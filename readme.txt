Rollen:
1. leerling
2. docent
99. SuperAdmin

Afspraken en regels:
leerlingen kunnen docenten niet als vriend hebben
docenten kunnen leerlingen wel als vriend hebben

SuperAdmin heeft geen vrienden, maar kan alle data uit database raadplegen

De pagina's Mijn Vrienden en Profiel kan alleen door de rollen 1 en 2 worden ingezien en alleen als je ingelogd bent.
De pagina Admin mag alleen door de SuperAdmin ingezien worden.

Mijn vrienden bevat een overzicht van alle vrienden van de gebruiker die ingelogd is.
Als je op een vriend klikt ga ja naar het profiel van die vriend.

De pagina Profiel is het profiel van de gebruiker die ingelogd is.

De pagina Admin is een overzicht van alle gebruikers.
